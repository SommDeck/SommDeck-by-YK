A digital wine list and stock management system developed by a senior sommelier who is also an otaku.The front-end wine list is based on data from a connected Google Sheet, while the stock management system functions as a two-way street, incorporating real-time stock tracking with detailed records. Built-in custom filters allow users to easily sort by grape varieties and custom tags across many types of wine and spirits. If a specific item or category is not needed, simply remove it from the Main List, and the back-end will automatically handle the rest.The purpose of this project is to streamline my own workflow and, ideally, make life easier for fellow sommeliers.As this project is developed for commercial purposes, please contact jogu20313@gmail.com if you are interested.
